# Terminal Calculator

A simple, retro-styled "Terminal" calculator built with HTML, CSS, and JavaScript.

## Features
- Neon green Matrix-style UI.
- Basic arithmetic operations (+, -, *, /).
- Full Keyboard Support (Numbers, Operators, Enter, Backspace).
- Error handling for invalid expressions.
- Mobile-responsive layout.

## How to Run
1. Clone this repository.
2. Open `index.html` in any modern web browser.

## Technologies Used
- HTML5
- CSS3 (Custom Styles)
- Vanilla JavaScript
